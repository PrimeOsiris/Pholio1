2.1P
Responsive Web design is design elements or features built into a web page that respond to certain pieces of data such as the size of the screen the page is being displayed on.
It is important because it makes the page more accessible to users which can increase profits/click through rate for the page/company.

2.2P
Implementing Responsive Web Design varies based on what type of web page you are creating. It can vary from implementing code that allows the page to adapt to different screen sizes to simply changing the size of the font.
